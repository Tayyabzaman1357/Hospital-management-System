⚠️ Error Detected: Cloud Firestore API has not been used in your project or is disabled.
ℹ️ Follow these 2 simple steps to enable Firestore:

1 Enable Firestore API

Click the button below to open the Google Cloud Console and enable the Firestore API:

🔗 Enable Firestore API (Click Here)

Click "ENABLE" button on the page that opens

2 Create Firestore Database

After enabling the API, create your database:

📁 Go to Firebase Console → Firestore

Click "Create database" → Choose "Test mode" → Select location → Click "Enable"

Check Status

  • ⏳ Checking Firestore API...
  • ⏳ Checking Database...
🚀 Launch Application

📝 Detailed Instructions:

  1. Click "Enable Firestore API" button above
  2. Click the ENABLE button on the Google Cloud Console page
  3. Wait for it to show "API Enabled"
  4. Click "Go to Firebase Console → Firestore" button
  5. Click "Create database"
  6. Choose "Start in test mode" (for development)
  7. Select a location closest to you
  8. Click "Enable"
  9. Wait 30-60 seconds for changes to propagate
  10. Come back here and click "Check Status"
  11. When all checks pass, click "Launch Application"
